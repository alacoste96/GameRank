Ayuda y Documentación

¿Qué es GameRank?

GameRank es una plataforma web para explorar, puntuar y seguir videojuegos free-to-play. La aplicación recoge datos de múltiples fuentes (XML y APIs JSON), los integra y permite a los usuarios autenticados interactuar con ellos.

¿Qué puedo hacer en GameRank?

  • Explorar una base de datos unificada de videojuegos.
  • Ver información detallada de cada juego.
  • Valorar cada juego con una puntuación del 0 al 5.
  • Escribir y leer comentarios sobre los juegos.
  • Seguir juegos que te interesen para acceder rápidamente a ellos.
  • Personalizar la interfaz: alias, tipo y tamaño de fuente.

Estructura de la aplicación

GameRank se basa en una arquitectura Django. Los juegos se almacenan en una base de datos local a partir de fuentes externas. Las vistas principales incluyen:

  • Página principal: Juegos ordenados por puntuación media.
  • Detalles del juego: Comentarios, puntuaciones, botón para seguir y valoración.
  • Votados: Juegos que has valorado.
  • Seguidos: Juegos que has seguido.
  • Configuración: Personaliza tu perfil y preferencias visuales.
  • Ayuda: Esta página informativa.
